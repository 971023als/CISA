# 5.11.3 IPS와 IDS, Firewall 비교

## 가. IPS와 IDS, Firewall 비교

| 구분       | Firewall                         | IDS                              | IPS                              |
|------------|----------------------------------|----------------------------------|----------------------------------|
| **목적**   | 접근통제, 인가, 내부망 보호       | 침입 여부 감지                   | 침입 이전에 방지                 |
| **특징**   | 수동적 방어                      | 로그, 시그니처 기반의 패턴 매칭  | 정책, 규칙 DB 기반의 비정상적 행위 탐지 |
| **패킷차단** | O                               | X                                | O                                |
| **오탐율** | X                                | O                                | O                                |
| **유실율** | O                                | O                                | X                                |
| **이상탐지** | X                                | O                                | O                                |

### 동작 기능
- **Firewall**: 전역 제어, 네트워크 접속
- **IDS**: 침입 탐지, 패턴 탐지, 네트워크 접속
- **IPS**: 침입 방지, 패턴 탐지, 네트워크 접속

### 장점
- **Firewall**:
  - 일정한 접근 제어
  - 네트워크 보호
- **IDS**:
  - 실시간 탐지
  - 사후 분석 대응
- **IPS**:
  - 실시간 탐지 및 방지 가능
  - 세션 기반 탐지 가능

### 단점
- **Firewall**:
  - 내부 사용자 공격 방지 어려움
  - 변형된 패킷 탐지 어려움
- **IDS**:
  - 오탐 발생 가능
  - 고가 장비 요구
- **IPS**:
  - 오탐 발생 가능
  - 고가 장비
