## 📝 시험 구조

- **총 문항:** 150문항  
- **시간:** 4시간  
- **5개 도메인 비중:**  
  1. 정보 시스템 감사 프로세스 (21%)  
  2. IT 거버넌스 및 관리 (17%)  
  3. 정보 시스템 획득, 개발 및 구현 (12%)  
  4. 정보 시스템 운영, 유지보수 및 서비스 관리 (23%)  
  5. 자산 보호 (27%)

## 💼 실무 적용 사례

### **1. 감사 프로세스 개선**
- **문제:** 조직 내 IT 자산 관리 프로세스의 미흡한 점 발견  
- **해결:** CISA 도메인 1(정보 시스템 감사 프로세스)에서 배운 체계를 활용해 프로세스를 문서화하고 감사 기준에 부합하도록 개선.  

### **2. 보안 통제 강화**
- **문제:** 비효율적인 접근 제어 정책  
- **해결:** 도메인 5(자산 보호) 학습 내용을 기반으로 새로운 접근 제어 정책 설계 및 도입.
